# 🚀 Employee Management System

A modern **Employee Management System** built using **Spring Boot**, **Spring Data JPA** and **MySQL**. The application provides RESTful APIs to perform complete CRUD (Create, Read, Update, Delete) operations on employee records.

---

## 🛠️ Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Maven

### Database
- MySQL

### Tools
- IntelliJ IDEA
- Postman
- Git
- GitHub
